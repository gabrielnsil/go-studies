# Go Studies

This repository contains my exercises, thoughts, and experiments while learning Go.

The goal is to build a solid foundation in Go by focusing on test-driven development (TDD), core language concepts, and idiomatic patterns.

## Modules
- **hello**
  - basic function creation and test
- **integers**
  - Simple arithmetic and introduction to writing tests
- **iteration**
  - Working with loops and simple string manipulation
- arrays
  - Handling collections, slices and aggregations.

## Requirements
- Go 1.20+
## References

- https://quill.gitbook.io/learn-go-with-tests/

## Notes
This is a study repository, so the focus is on learning and experimentaion rather than production-ready code.
